Učitel

Země se točí,
                         říká žáček.
Nikoli, země se točí,
                         říká učitel.

Hory se zelenají,
                         říká žáček.
Nikoli, hory se zelenají,
                         říká učitel.

Dvakrát dvě jsou čtyři,
                         říká žáček.
Dvakrát dvě jsou čtyři,
                         opravuje ho učitel.

Protože učitel to ví líp.
